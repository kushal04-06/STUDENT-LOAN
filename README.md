# 💰 Loan Prediction

A modern, interactive web application that helps users **predict loan eligibility**, analyze financial data, and manage loan-related insights through a visually appealing dashboard.

## 📌 Project Overview

This project is a **Loan Prediction System + Dashboard UI** designed with modern UI/UX principles. It allows users to:

* Predict loan eligibility
* Visualize financial insights
* Interact with dynamic UI components
* Experience smooth animations and 3D effects

---

## ✨ Features

### 🔐 Authentication System

* Login Page with modern UI
* Forgot Password module
* Secure input handling

---

### 📊 Loan Prediction Module

* User input-based loan eligibility prediction
* Credit score visualization
* Interactive result display

---

### 📈 Dashboard

* Financial insights overview
* Dynamic UI with animations
* Clean and responsive layout

---

### 🎨 UI/UX Design

* Modern gradient background
* Glassmorphism effects
* Animated blob backgrounds
* 3D tilt card interaction
